## 工具调用接口

### `generate_title(drama_info, region, preset)`

**输入**：
```json
{
  "drama_info": {
    "name": "剧名",
    "genre": "题材",
    "hooks": ["钩子1", "钩子2"],
    "key_props": ["道具1"],
    "characters": ["男主", "女主"]
  },
  "region": "hk|tw|sg|mo|id|br|vn",
  "preset": "fast_validation|full_rebuild",
  "count": 3
}
```

**输出**：
```json
{
  "candidates": [
    {
      "title": "标题文本",
      "score": {"hook": 80, "conflict": 90, "curiosity": 70, "alignment": 85, "total": 83.5},
      "tags": ["#短劇", "#flickreels"],
      "compliance_check": {"has_drama_keyword": true, "length_ok": true, "conflict_count": 3}
    }
  ],
  "rejected": ["不通过的标题及原因"]
}
```

**规则卡执行顺序**：
1. SDE-017：检查含「短劇/Drama」→ REJECT 否则
2. SDE-001：检查≥3个冲突点 → REJECT 否则
3. SDE-004：检查长度（繁体30-60字/英文80-100字符）→ WARN 否则
4. SDE-016：如为合集检查含「全集/完整版」→ WARN 否则
5. SDE-018：检查题材优先级匹配度 → WARN 否则

### `generate_cover(title, analysis, preset)`

**输入**：
```json
{
  "title": "选定标题",
  "analysis": {"characters": [], "key_scenes": [], "key_props": [], "hooks_and_twists": []},
  "preset": "fast_validation|full_rebuild",
  "region": "hk"
}
```

**输出**：
```json
{
  "candidates": [
    {
      "brief": "中文描述给设计师",
      "prompt": "英文AI生图prompt",
      "elements": ["人物", "道具", "情绪", "场景", "身份"],
      "score": {"total": 88, "match_title": 90, "emotion": 85, "clutter": 95},
      "color_scheme": "金色/黑色"
    }
  ]
}
```

**规则卡执行顺序**：
1. SDE-007：检查≥3个元素（人物+道具+情绪）→ REJECT
2. SDE-008：检查≤5个元素 → WARN
3. SDE-009：色彩匹配题材 → WARN
4. SDE-010：5项检查≥4项通过 → REJECT

## 决策启发式

1. **先发再选地区**：不要先选好地区再发，先发再看数据选地区
2. **3+冲突点标题**：标题必须包含至少3个冲突点/钩子/反转
3. **前3幕关键元素封面**：封面必须包含前3幕的关键元素
4. **合集必须标"全集"**：>1h视频标题或描述必须含"全集/完整版"
5. **标题必须含题材标识**：用户3秒内知道是什么类型
6. **每天1-2部更新**：保持稳定更新频率
7. **爆款加倍复制**：爆款出现后，同类题材加倍
8. **合规检查必做**：上传前必须做合规检查
